# KotlinBasics
## kotlinとは
InteliJ IDEA というIDEを開発しているJetBrains社が開発したプログラミング言語。
Java バイトコードにコンパイルされるため、JVM上で動作する。
静的型付け言語で、Javaと完全な互換性がある。
JavaのコードをKotlinに変換することも可能。

## コーディングスタイル
変数名や関数名はキャメルケース（camelCase）を使用する。
クラス名は大文字で始める、パスカルケース（PascalCase）を使用する。
関数名やプロパティ名は小文字で始める。
インデントはスペース4つを使用する。
public関数にはKDocコメントを使用する。
